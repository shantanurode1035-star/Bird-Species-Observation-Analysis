# Bird Species Observation Analysis

## Project Overview
Analysis of bird species across Forest and Grassland ecosystems
across 11 National Parks (2018)

## Files
- cleaned_combined.csv — Cleaned dataset
- Bird_Species_EDA.ipynb — EDA notebook
- Bird_Species_Analysis.ipynb — Species Analysis
- Bird_Species_SQL.ipynb — SQL Analysis
- Bird_Species_Visualization.ipynb — Visualizations
- Bird_Species_Dashboard.pbix — Power BI Dashboard

## Key Findings
- Total Observations: 15,368
- Unique Species: 126
- Most Common: Northern Cardinal
- Most Biodiverse Park: MONO
